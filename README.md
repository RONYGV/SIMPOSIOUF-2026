# III Simposio — paquete completo para GitHub Pages

## Cómo publicarlo
Sube **todo el contenido de esta carpeta** a la raíz del repositorio utilizado por GitHub Pages:

- `index.html`
- carpeta `assets/`
- `.nojekyll`

No cambies los nombres ni la estructura de las carpetas, porque la página utiliza rutas relativas para cargar las imágenes.

## Corrección aplicada
- Se eliminó el botón personalizado grande de reproducción en dispositivos móviles.
- Se eliminó el texto “Toca para reproducir”.
- El reproductor de YouTube se muestra directamente con sus controles.

## Estructura
```
index.html
assets/
  img/
  manifest.csv
  manifest.json
.nojekyll
README.md
```
